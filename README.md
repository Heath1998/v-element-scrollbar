# v-element-scrollbar
从element-ui分离出来的scrollbar
## Install
```
$npm install v-element-scrollbar
```
## 使用

```
// main.js
import Vue from 'vue'
import vElementScrollbar from 'v-element-scrollbar'
Vue use(vElementScrollbar)
// **.vue
<template>
    <div>
        <v-element-scrollbar>
            <div>内容</div>
        </v-element-scrollbar>
    </div>
</template> 
```
